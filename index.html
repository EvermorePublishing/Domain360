Server360 Sites

Server360 Sites is our website hosting service that lets you host your own personal webpage using our Domain360 GitHub repo. We use this repo along with Cloudflare Pages to run our service, which is why the main URL is https://server360sites.pages.dev/.

To use this service, do the following:

  • Fork the Domain360 GitHub repo
  • Create a new folder with the name of the URL slug you want (make sure the name is unique.)
  • Submit a merge request.

If we accept to host your minisites, we will notify you once it is up and running.
